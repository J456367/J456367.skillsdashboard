    Tensorflow & Django - Name Classification

    Tensorflow & Django - Name Classification from James Connolly on Vimeo .

    This is a deep learning model which has been tested in the Django web application frame work, the model takes in text data and one-hot encodes the text data before the data is passed through a tensor flow deep learning network. The network has been accurately tested names outside the test and training data.

    Challenges involved in this project included a workaround with asynchronous event handling which was incorporated Python's Celery library.
