Github - Sudoku Solver

This script will solve a number of Suduko problems from easy to medium puzzles.
The script is an algorithmic approach to solving the problem and noting making use of a brute force solution.
The algorithm will continue to loop until complete or two iterations of the same loop without changes.

As an on going project to work on my functional programming skills. This software create a 9 x 9 matrix as input.
A 9 x 9 x 9 tensor for validating potential algorithmic options and returns a 9 x 9 matrix as an output.
As more functions are added more difficult solutions can be solved.
