    Technology Skills Dashboard

    Python

    Python is an interpreted high-level general-purpose programming language. Python's design philosophy emphasizes code readability with its notable use of significant indentation.

    Pandas

    Pandas is a software library written for the Python programming language for data manipulation and analysis. In particular, it offers data structures and operations for manipulating numerical tables and time series.

    Numpy

    NumPy is a library for the Python programming language, adding support for large, multi-dimensional arrays and matrices, along with a large collection of high-level mathematical functions to operate on these arrays.

    SQL

    SQL (pronounced "ess-que-el") stands for Structured Query Language. SQL is used to communicate with a database.

    Tensorflow

    TensorFlow is a free and open-source software library for machine learning. It can be used across a range of tasks but has a particular focus on training and inference of deep neural networks. Tensorflow is a symbolic math library based on dataflow and differentiable programming.

    Plotly

    Plotly is a scientific graphing tool for analytics, modeling and statistics. The plots on this dashboard are designed with the tool.

    Google Cloud Compute

    GCP is a cloud computing platform by Google.

    JavaScript

    JavaScript is a high level programming language used in web development, modern browsers have JS built in.

    HTML

    The HyperText Markup Language, or HTML is the standard markup language for documents designed to be displayed in a web browser.

    CSS

    Cascading Style Sheets is a style sheet language used for describing the presentation of a document written in a markup language such as HTML.
