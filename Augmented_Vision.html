    Augmented Vision - Data Engineering

    Augment Vision from James Connolly on Vimeo .

    This software takes a photograph as input, resizes the image dependant on the maximum value of the variable N then converts the photograph into a list of colours. N defines the number of Unicode characters used for a caluclation. Values between 1 and 1024 can be used. A pattern can be selected which will use the Unicode characters to create a 2d matrix. The object is then tiled and displaced upon itself several times depending on the user defined resolution.

    The python script will then associate a colour with a Unicode value and iterate through all pixels in the image until the image has been converted for a 2d unicode matrix to a 3d rgb matrix. The image will then be printed into a user defined format such as TIFF or JPEG dependant on the the compression requirements and place in a directory.

    If the user has selected the option to make video, the list of colours will rotate one position forwards for each frame of the image. While the option for the variable N to change will allow the size of the base matrix to change. The process listen in the first paragraph starts over for each new frame.
